This homework assignment involves collaborative work to develop a brand name and create SVG icons representing the different flavors we offer. The SVG icons are designed to interact or animate upon hovering. Additionally, JavaScript functionality has been added to each SVG icon to list all the flavors. When an icon is clicked, the corresponding flavor name will be logged to the browser's console, viewable via the Inspect Element tool.
